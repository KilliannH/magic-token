## рџ’Ў TIPS

1. **Consistency**: Generate one wizard first, then use it as a style reference
   for the others (PixelLab supports reference images)
2. **Animations**: Once you have the idle sprites, use PixelLab's animation
   tool to create 2-4 frame idle animations and attack animations
3. **Batch**: Generate all fire-related assets together for consistent style,
   then move to ice, etc.
4. **Cleanup**: Use PixelLab's inpainting tool to fix any artifacts
5. **Size**: The game renders characters at ~80-100px, so 64x64 source will
   look great with pixel-perfect scaling (image-rendering: pixelated)
